# Emmi Issue Tracker

Author: Liam Mills

To start the app:
1. Run ```yarn install```
1. Run ```yarn json-server --watch db.json --port 8000```
1. In a new terminal, run ```yarn start``` and open [http://localhost:3000](http://localhost:3000).
